# Python Project

## Project Description
This is a basic Python project structure with:
- Main script (geneL)
- Requirements file
- README documentation

## Setup Instructions
1. Install Python 3.15.0b2
2. Create virtual environment:
   ```powershell
   python -m venv venv
   ```
3. Activate environment:
   ```powershell
   venv\Scripts\Activate.ps1
   ```
4. Install dependencies:
   ```powershell
   pip install -r requirements.txt
   ```
5. Run the project:
   ```powershell
   python geneL
   ```
