# Lovable AI Project

AI-powered design and development with Lovable + Claude Pro

## Setup

### 1. Connect GitHub Integration
- Go to: https://lovable.dev
- Click "New Project"
- Select GitHub repository: `umittopcuoglu/genel`
- Select branch: `main`
- Set folder: `lovable/`

### 2. Configure Claude Integration
- Settings → Integrations → Claude
- Add Claude Pro API keys:
  - Koray: sk-ant-api03-...
  - Umit: sk-ant-api03-...

### 3. Start Designing
- Open Lovable dashboard
- Create new designs
- Auto-commit to GitHub
